### Decision Rulebook for IEEE 118-Bus Voltage Prediction

Based on the analysis of historical test cases and system behavior, the following decision rules can be used to estimate voltage magnitudes ($vm_{pu}$) without running a full power flow simulation.

### Rule Category 1: Generator Bus Identification (Stiff Voltage)
These buses are connected to large generators with Automatic Voltage Regulators (AVRs). They maintain a constant voltage regardless of system loading, only dipping slightly under extreme stress.

*   **IF** the bus ID is in the set **{4, 5, 6, 8, 9, 12, 24, 25, 26, 49, 59, 66, 80, 89, 90, 100, 106, 110, 112, 116}**
    *   **THEN** predict $vm_{pu} \approx 1.200$.
    *   *Refinement:* If total system load > 11,500 MW, predict $vm_{pu} \approx 1.190$ to account for minor reactive stress.

### Rule Category 2: System Loading Conditions (Global Voltage Level)
The total active power load ($load\_p\_total\_mw$) is the primary driver for voltage depression across the system. Use Bus 1 as the primary indicator for the system-wide voltage state.

*   **IF** $load\_p\_total\_mw < 7,000$ MW (Low Load / Off-peak)
    *   **THEN** predict **Bus 1** voltage $\approx 1.130 - 1.135$ pu.
*   **IF** $7,000 \le load\_p\_total\_mw \le 9,500$ MW (Medium Load)
    *   **THEN** predict **Bus 1** voltage $\approx 1.110 - 1.125$ pu.
*   **IF** $9,500 < load\_p\_total\_mw \le 11,000$ MW (High Load)
    *   **THEN** predict **Bus 1** voltage $\approx 1.090 - 1.105$ pu.
*   **IF** $load\_p\_total\_mw > 11,000$ MW (Extreme Peak Load)
    *   **THEN** predict **Bus 1** voltage $\approx 1.065 - 1.085$ pu.

### Rule Category 3: Time-of-Day Patterns (Load Proxy)
When load data is unavailable, use the timestamp to infer the loading state and apply the corresponding voltage levels from Category 2.

*   **IF** Hour is **00:00 - 04:00** (Deep Night)
    *   **THEN** Assume **Low Load** state (Bus 1 $\approx 1.13$ pu).
*   **IF** Hour is **09:00 - 15:00** (Daytime)
    *   **THEN** Assume **Medium Load** state (Bus 1 $\approx 1.11$ pu).
*   **IF** Hour is **16:00 - 20:00** (Evening Peak)
    *   **THEN** Assume **High Load** state (Bus 1 $\approx 1.09$ pu).

### Rule Category 4: Voltage Profile Estimation (Bus Relationships)
Once the voltage of Bus 1 is estimated using Categories 2 or 3, the voltages of other major load buses can be derived using fixed offsets, as the network topology preserves relative voltage differences.

*   **IF** Bus 1 voltage is known ($V_1$)
    *   **THEN** predict **Bus 2** voltage $\approx V_1 + 0.025$ pu.
    *   **THEN** predict **Bus 3** voltage $\approx V_1 + 0.015$ pu.
*   **IF** Bus 1 voltage is known ($V_1$)
    *   **THEN** predict **Bus 4** (Generator) voltage $\approx 1.195$ pu (independent of $V_1$).
    *   **THEN** predict **Bus 5** (Generator) voltage $\approx 1.195$ pu (independent of $V_1$).

### Rule Category 5: Seasonal and Solar Effects
*   **IF** Month is **July or August** (High Summer Demand)
    *   **THEN** Apply a **-0.01 pu** correction to the base load prediction (e.g., if Medium Load predicts 1.12, predict 1.11) due to increased thermal stress and air conditioning load.
*   **IF** Month is **November, December, or January** (Winter Heating)
    *   **THEN** Apply a **-0.005 pu** correction to the base load prediction.
