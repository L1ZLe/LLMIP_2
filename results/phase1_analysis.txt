### 2. Key Relationships and Feature Thresholds

The primary driver of voltage variation on Category B and C buses is the **Total System Active Power Load (`load_p_total_mw`)**.

#### Relationship 1: Total Load vs. Weak Bus Voltage
There is a strong inverse correlation between total load and voltage at weak buses (e.g., Bus 21, Bus 22).

*   **Low Load Condition:** `load_p_total_mw` < 7,000 MW
    *   **Effect:** Weak bus voltages are high (> 1.12 pu).
    *   *Example:* At `load_p_total_mw` = 6415 MW, `vm_21` = 1.129 pu.
*   **High Load Condition:** `load_p_total_mw` > 11,000 MW
    *   **Effect:** Weak bus voltages drop significantly (< 1.07 pu).
    *   *Example:* At `load_p_total_mw` = 12,356 MW, `vm_21` = 1.040 pu.

#### Relationship 2: Time of Day vs. Load
Temporal features are strong proxies for load levels.
*   **Peak Hours:** 17:00 – 20:00 (Hour 17, 18, 19, 20)
    *   Correlates with highest `load_p_total_mw` and lowest voltages.
*   **Off-Peak Hours:** 02:00 – 05:00 (Hour 2, 3, 4, 5)
    *   Correlates with lowest `load_p_total_mw` and highest voltages.

#### Relationship 3: Renewable Generation (Solar)
*   **Solar Impact:** High `gen_solar_p_r1_mw` (Region 1 Solar) during midday (Hours 10-14) provides a voltage boost to Region 1 buses (1-30), slightly counteracting the load increase compared to evening peaks where thermal generation dominates.

### 3. Conditional Decision Rules (IF-THEN)

These rules can be applied to predict voltage magnitudes for new data points:

#### Rule Set 1: Predicting Weak Bus Voltages (e.g., Bus 21, 22, 115)
*   **IF** `load_p_total_mw` **>** 12000:
    *   **THEN** `vm_21` **<** 1.05 pu (Critical Low)

*   **IF** `load_p_total_mw` **<** 7000:
    *   **THEN** `vm_21` **>** 1.12 pu (High)

*   **IF** `hour` **IN** [17, 18, 19, 20] **AND** `is_weekend` == 0:
    *   **THEN** Expect `vm_21` to be near its minimum for the day (Voltage Sag).

#### Rule Set 2: Predicting Generator Bus Voltages

    *   **THEN** `vm_pu` = 1.20 pu (Constant)

#### Rule Set 3: Predicting Standard Load Bus Voltages (e.g., Bus 1, 2, 3)
*   **IF** `load_p_total_mw` **>** 11000:
    *   **THEN** `vm_1` is approx 1.08 pu.
*   **IF** `load_p_total_mw` **<** 7000:

*   **IF** `gen_solar_p_r1_mw` **>** 600:

#### Rule Set 4: Regional Load Impact
*   **IF** `load_p_r3_mw` **>** 3500 MW:

### Summary for Prediction
To predict voltages for a new sample:
1.  **Check Bus Type:** If it is a Generator Bus (Category A), predict **1.20 pu**.
2.  **Check Total Load:** Use `load_p_total_mw` as the primary regressor.
    *   High Load (>11k MW) $\rightarrow$ Subtract ~0.05 pu from base voltage.
    *   Low Load (<7k MW) $\rightarrow$ Add ~0.03 pu to base voltage.
3.  **Check Weak Buses:** Apply a steeper penalty for high load on Buses 21, 22, 23, 115, 117.
4.  **Check Solar:** If solar is high in a region, slightly boost voltage predictions for that region.
